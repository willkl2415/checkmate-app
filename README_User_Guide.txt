
Checkmate: Local Windows Installation Guide

1. Double-click `run_checkmate.bat` to start Checkmate.
2. Ensure Python 3.11 is installed.
3. Place your DOCX files in the `source_docs` folder.
4. Run `ingest.py` before using `app.py` if new documents are added.
5. Visit http://127.0.0.1:5000 in your browser.

This version is fully offline, safe, and expandable to handle thousands of documents.
